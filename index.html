Sync Across Devices

Your scores sync through a private GitHub gist — free, nothing to host, no database. Set it up once per device with a GitHub token.

  1. Create a token on GitHub (the gist scope is pre-selected — leave everything else off)
  2. Press Generate token, copy it, and paste it below.

Scores push automatically after every change and pull whenever you open or return to the app. If two devices edit at the same time, the most recent save wins.
